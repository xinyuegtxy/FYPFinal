Introducing Analyzer

Welcome to the Analyzer section. The Analyzer tools are available at the sidebar.

Paper-Based Tools

Tag Cloud

Tag Cloud is a tool to visualize the frequencies of tags within a specific paper. Tag can be keyword-tag, concept-tag, or formula-tag.

Topic Distribution

Topic Distribution is a visualization of the marks/question distribution within a specific paper.

Topic Trend

Topic Trend is a visualization of topic distribution trend within a specific historical periods.

Concept Distribution

Concept Distribution is a visualization of the concept occurrence distribution within a specific paper.

Concept Trend

Concept Trend is a visualization of concept occurence distribution trend within a specific historical periods.

Topic-Based Tools

Tag Cloud

Tag Cloud is a tool to visualize the frequencies of tags within a specific topic. Tag can be keyword-tag, concept-tag, or formula-tag.

Question Clusters

Question Clusters is a data mining clustering tool which runs on K-Means algorithm to spot the similarities of questions into clusters based on tags as the dimensions of measure.
